# Goormpen - 실시간 채팅 개인 프로젝트


## 소개
- 리액트와 스프링을 연동해 배포하였습니다.
- 웹소켓을 이용한 실시간 채팅 서비스입니다.

## 스택

- React, sockjs-client, stompjs
- Spring boot, Websocket
- AWS EC2

## 기능

- 전체 채팅 방
- 1대1 메세지 방
